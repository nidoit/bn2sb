# Blunux 셀프빌드 - 한눈에 보는 초간단 설명

> README 파일의 내용을 아주 쉽게 설명해드릴게요.

---

## 이 프로젝트가 뭐예요?

**Blunux**는 **나만의 리눅스 운영체제(OS)를 직접 만들 수 있는 도구**입니다.

쉽게 말하면:
- 윈도우처럼 컴퓨터를 켜면 나오는 운영체제를 **내가 직접 만드는 것**
- 한글이 기본으로 들어있고, 원하는 프로그램도 미리 넣을 수 있음
- 만든 결과물은 **ISO 파일** (USB에 담아서 아무 컴퓨터에 설치 가능)

---

## 전체 과정을 7단계로 정리하면:

### 1단계: 준비하기
- Arch Linux 계열 컴퓨터가 필요해요 (없으면 VirtualBox에 Manjaro 깔면 됨)
- 인터넷, 저장공간 20GB, USB 8GB 이상 필요

### 2단계: 필요한 프로그램 설치
- 터미널(검은 화면 창)을 열고 명령어를 복사-붙여넣기하면 끝
- `sudo pacman -Syu` → 시스템 업데이트
- `sudo pacman -S archiso base-devel git cmake --needed` → 도구 설치
- Julia라는 프로그래밍 언어도 설치 (빌드 도구가 이걸로 동작함)

### 3단계: Blunux 다운로드
- `git clone` 명령어로 GitHub에서 소스코드를 받아옴
- 그냥 "다운로드 버튼" 같은 거라고 생각하면 됨

### 4단계: ISO 만들기 (핵심!)
- `sudo julia build.jl config_kr.toml` 한 줄이면 자동으로 만들어짐
- 20분~1시간 기다리면 ISO 파일 완성
- `config_kr.toml` = 한국어 설정 파일 (여기서 프로그램 추가/제거 가능)

### 5단계: USB에 굽기
- 만들어진 ISO 파일을 USB에 복사하는 과정
- `dd` 명령어를 사용 (USB 장치 이름을 잘 확인해야 함!)
- **주의:** USB 안의 기존 데이터는 다 날아감

### 6단계: 컴퓨터에 설치하기
- USB 꽂고 → 컴퓨터 켜면서 부팅키(F2, F12 등) 누르기 → USB 선택
- `blunux-install` 명령어 입력하면 설치 프로그램이 알아서 진행

### 7단계: 문제 해결
- 에러가 나면 대부분 `sudo`를 빼먹었거나 Julia가 안 깔린 경우
- USB 부팅이 안 되면 BIOS에서 Secure Boot 끄기

## 설정 파일(`config_kr.toml`)로 할 수 있는 것

```toml
# 예시: 원하는 프로그램을 true로 켜면 ISO에 포함됨
[packages.browser]
firefox = true    # 파이어폭스 넣기
chrome = true     # 크롬 넣기

[packages.office]
libreoffice = true  # 리브레오피스 넣기
```

**프로그램 추가 방법:** 설정 파일 열고 → `프로그램이름 = true` 추가 → 다시 빌드

## 보안 주의사항

기본 비밀번호가 `root` / `user`로 되어있는데, 이건 **테스트용**이에요.
실제로 쓸 컴퓨터에 설치했다면 **반드시 `passwd` 명령어로 비밀번호를 바꾸세요!**
